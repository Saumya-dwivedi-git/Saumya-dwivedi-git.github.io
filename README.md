# Saumya Dwivedi — GitHub Portfolio

A responsive, single-page developer portfolio ready for GitHub Pages.

## Personalize it

Open `script.js` and replace the two placeholders at the top:

- `YOUR_GITHUB_USERNAME`
- `YOUR_LINKEDIN_USERNAME`

Review the repository name, project text, email, and skill list before publishing.

## Preview locally

Open `index.html` directly, or run:

```powershell
python -m http.server 8000
```

Then visit `http://localhost:8000`.

## Publish with GitHub Pages

1. Create a repository named `your-username.github.io`.
2. Upload this folder contents to the repository root.
3. Open **Settings → Pages**.
4. Select **Deploy from a branch**.
5. Select `main` and `/ (root)`, then save.
6. Visit `https://your-username.github.io`.

## Included

- Responsive desktop and mobile layout
- Dark/light theme with saved preference
- Accessible navigation and reduced-motion support
- About, capabilities, projects, approach, and contact sections
- Editable GitHub, LinkedIn, email, and project links
- No build tools or framework required
